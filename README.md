# kibaco用Chrome拡張機能
この拡張機能はkibacoへ課題提出時にPDFのプレビューを提出確認画面に表示させます。

インストールは[こちら](https://chrome.google.com/webstore/detail/kibaco-advance/gcallmgoceccjabpmdljlkfblgolbakg/related?hl=ja&authuser=0)から

Chromeストアへの公開は現在審査中です。
